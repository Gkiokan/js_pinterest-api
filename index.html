Pinterest API Wrapper

by Gkiokan Sali | Gkiokan.NET

Welcome to my Pinterest API Wrapper

Today I'll show you, how you can use your own or whoevers Pinterest Account
for getting the Pins from a specific Board and show them on your page.

Current Data which are provided is my own one, Username: Gkiokan1337 and Board Ladies.

How To use this and make some magic :)

1. Create the Wrapper

We create a div with a specific class, which will contain all our Pins, lets call it the-magic-wrapper.

2. Call the Method

After we have declared our Wrapper - where obviosly all the Pins will be filled in - we just need to trigger the Pinterest API to get some magic.
In my case i will write following js code:
g.pinterest.getBoard('gkiokan1337/ladies', 'myTokenWhatever', '.the-magic-wrapper');
